# node-gridfs-toolkit
The easy way for MongoDB Gridfs

# 参数列表

|名称|关键字|别名|说明|
|-|-|-|-|
|帮助|-h|--help|获取帮助列表|
|主机地址|-host|无|设置数据库的主机地址|
|端口|-port|无|设置数据库的端口|
|用户名|-u|--user|设置数据库的用户名|
|用户密码|-p|--password|设置数据库的用户密码|
|认证数据库|-authenticationDatabase|无|设置数据库的用户认证数据库|
|数据库名称|-d|--db, --database|设置所要获取的数据库的名称|
